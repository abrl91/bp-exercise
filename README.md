# bp-exercise

Requirements: 
* node version 14 or greater

Dependencies: 
* express
* mnemonist
* test: jest

Dev-ependencies:
* nodemon
* jest
* @babel/preset-env

Running on develop: 

```
npm run dev
```
Running without nodemon

```
npm run start
```

Tests
```
npm run test
```

apis:
* get eventsCount: http://localhost:8081/eventsCount
* get wordsCount: http://localhost:8081/wordsCount
* get last60Sec: http://localhost:8081/last60Sec
